HTML Tags

  1. <header>:Defines a header for a document or section.
  2. <footer>:Defines a footer for a document or section.
  3. <img>:Defines an image
  4. <link>:Defines the relationship between a document and an external resource (most used to link to style sheets).
  5. <main>:Specifies the main content of a document.
